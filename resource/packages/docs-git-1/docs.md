# 初めに

## Gitとは

Gitは、分散型バージョン管理システムの一つです。

バージョン管理システムとは、ファイルの変更履歴を管理するためのシステムです。
Gitを使うことで、ファイルの変更履歴を管理し、過去の状態に戻したり、複数人での開発を行う際にお互いの変更を統合することができます。
